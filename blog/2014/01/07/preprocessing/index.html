Preprocessing
Jan 7th, 2014

I ran the preprocessing with this dataset. The full config file can be seen here: (https://github.com/czarrar/emotional-bs/blob/master/10_preprocessing/20_run/run_config.yaml)[https://github.com/czarrar/emotional-bs/blob/master/10_preprocessing/20_run/run_config.yaml]. The output can be found in /home2/data/Projects/Emotional-BS/processed_data. The working directories were unfortunately removed due to being run on gelert so you can’t regenerate certain things in a flash.

Here are some relevant preprocessing decisions:

  • I used bbreg
  • Anatomical -> Standard registration was done with ANTs
  • The functional data were registered to 2mm standard space
  • I applied two types of correction strategies (basically compcor with and without global):
    • compcor+motion+linear+quadratic
    • compcor+global+motion+linear+quadratic
  • Bandpass filtering of 0.01-0.1Hz was or wasn’t applied

I’m thinking for the actual analysis, I’ll use the non-filtered data. Since some of this is also task data, I should have possibly only used a high pass filter.

ROI Time-Series

I also extracted the time-series from several sets of parcellations:

  • AAL (anatomical)
  • CC200 (functional)
  • CC400 (functional)
  • EZ (anatomical)
  • HO (anatomical)
  • TT (anatomical)
  • ROIs from some paper (functional)

Since these masks are actually in 3mm space, the data was likely resampled so it is really unclear why I set the data to be registered to 2mm standard space.

Dual Regression

I used some ICA components and applied dual regression to them. These components can be found at /home2/data/Projects/ABIDE_Initiative/CPAC/abide/templates/3mm_for_dualreg/PNAS_Smith09_rsn10.nii.
